SOMOS O GRUPO BUG'S LIFE

O Voya-Go é um aplicativo inovador que atua como um guia turístico digital, projetado para proporcionar experiências de viagem personalizadas e inesquecíveis. Com ele, os usuários podem planejar, explorar e aproveitar destinos de forma prática, tudo na palma da mão. O Voya-Go simplifica o processo de organização de viagens, oferecendo informações detalhadas sobre locais turísticos, dicas de roteiros e sugestões personalizadas com base nos interesses de cada viajante.
"Grupo UPX: juntos desde o 1º semestre, estamos mais uma vez prontos para demonstrar nosso desempenho e criatividade na UPX IV. Let's Go!"

Confira o desenvolvimento do APP

O Voya-Go tem como missão tornar as viagens acessíveis e inclusivas para todos.
Nosso aplicativo oferece uma experiência turística única, permitindo que os usuários descubram novos destinos de forma independente.
Além disso, o Voya-Go se dedica a integrar pessoas com deficiência auditiva, oferecendo suporte em tempo real e recursos que as empoderam a explorar o mundo com autonomia, eliminando barreiras e promovendo a inclusão em cada jornada.

Tela incial de login, onde o usuário deverá autenticar seu usuário para ter acesso ao APP.

Tela de cadastro, será necessário cadastrar usuário o mesmo será enviado para o Banco de Dados Relacional no caso iremos trabalhar com o MYSQL

Dashboard será integrado a uma API de mapa GPS Online, está parte ainda se encontra em desenvolvimento!
